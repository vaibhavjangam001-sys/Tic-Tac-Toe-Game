## 📌 Project Overview

This is a browser-based **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.
The project implements core game logic, turn-based interaction, win detection, and UI updates dynamically.

Players take turns placing **X** and **O** on a 3×3 grid.
The game automatically detects a winner or a draw and displays the result.

This project helped me understand **DOM manipulation, event handling, and game logic implementation in JavaScript**.

---

## 🚀 Features

* Two-player Tic Tac Toe game
* Turn indicator for Player X and Player O
* Win detection (rows, columns, diagonals)
* Draw detection
* Reset game functionality
* New game option after winning
* Winning popup with trophy image
* Hover animation on boxes
* Game rules panel
* Responsive centered layout
* Deployed using **GitHub Pages**

---

### 3. Game Logic Implementation

I implemented logic to detect a winning pattern.

Example patterns:

```
[0,1,2]
[0,3,6]
[0,4,8]
[1,4,7]
[2,5,8]
[2,4,6]
[3,4,5]
[6,7,8]
```

The program checks whether three positions contain the same symbol.
